# TRPG キャラクターシート作成画面 概要
　以下にキャラクターシートを作成する画面において、入力項目とキャラクター自動生成におけるフローについてまとめる。

## キャラクターシート　項目構成
以下の要素で構成される

1. 基本情報
2. 能力値
3. 技能

### 基本情報

- 名前
- 職業*1
- 年齢
- 性別
- 身長
- 体重
- 出身
- 背景情報
- 行動パターン(性格的なもの)

### 能力値

- **ダイスにより決定するもの**

| 名称 (英語)        | 名称 (日本語) | サイコロの振り方 |
| ------------------ | ------------- | -------------- |
| strength         | 筋力          | 3d6            |
| constitution       | 体力          | 3d6            |
| power              | 精神力        | 3d6            |
| dexterity          | 機敏          | 3d6            |
| appearance       | 外見的な魅力 | 3d6            |
| size               | 体格          | 2d6+6          |
| intelligence       | 知性          | 2d6+6          |
| education          | 教養          | 3d6+3          |

- **ダイスにより算出されたパラメータから計算するもの**

| 名称 (英語)             | 名称 (日本語)   | 計算式          |
| ----------------------- | ------------- | ------------- |
| san                    | 正気度        | power * 5     |
| fortune                | 幸運          | power * 5     |
| idea                   | アイデア      | intelligence * 5 |
| knowledge              | 知識          | education * 5   |
| durability             | 耐久力        | (constitution + size) / 2 |
| magic\_point            | マジックポイント | power * 1     |
| vocational\_skill\_points | 職業技能ポイント | education * 20 |
| hobby skill points| 趣味技能ポイント| intelligence * 10  |
| damage bonus           | ダメージボーナス | strength + size |


### 技能

### 戦闘技能

| 技能名         | 初期値 | 説明|
| :------------- | :----- | :-------------------------------------------------------------------------------------------------------------------------------- |
| 回避           | DEX×2% | 攻撃や危険を目視できた場合に、それを避けるための能力。受け流しと合わせて1ラウンドに1回使用できるのが基本。           |
| キック         | 25     | 足による蹴り攻撃の能力。マーシャルアーツ技能と組み合わせることでダメージが増加することがある（6版）。                           |

| こぶし(パンチ) | 50     | 拳による殴り攻撃の能力。マーシャルアーツ技能と組み合わせることでダメージが増加することがある（6版）。                           |
| 頭突き         | 10     | 頭による攻撃の能力。マーシャルアーツ技能と組み合わせることでダメージが増加することがある（6版）。                             |

| 拳銃           | 20     | 拳銃（ハンドガン、リボルバーなど）を扱う能力。                                                                                       |

### 交渉技能

| 技能名                                 | 初期値  | 説明                                                                   |
| :------------------------------------- | :------ | :--------------------------------------------------------------------- |
| 言いくるめ                             | 5       | 口先で相手を騙したり、丸め込んだりする能力。                   |
| 信用                                   | 15      | 相手に自分を信用させる、または社会的信用度を示す能力。       |
| 説得                                   | 15      | 理論や感情に訴えかけて相手を納得させ、行動を変えさせる能力。         |
| 値切り                                 | 5       | 商品やサービスの価格交渉を行う能力。                           |
| 母国語（具体的な言語を記載、例：日本語） | EDU×5% | 母国語の読み書きや会話能力。通常は初期値で十分とされることが多い。 |
| 他の言語（具体的な言語を記載）           | 1%      | 母国語以外の言語の読み書きや会話能力。                              |

### 知識技能

| 技能名       | 初期値 | 説明                                                                                                                 |
| :----------- | :----- | :------------------------------------------------------------------------------------------------------------------- |
| 医学         | 5      | 人体の構造、病気、怪我に関する知識と診断、治療を行う能力。応急手当より専門的。                                   |
| オカルト     | 5      | 神秘主義、超常現象、民間伝承など、一般科学では説明できない分野の知識。                                       |
| 化学         | 1      | 物質の構造、性質、反応に関する知識。化合物の分析や生成など。                                                  |
| クトゥルフ神話 | 0      | 宇宙的恐怖や神話的存在に関する禁断の知識。通常プレイ開始時は0で、魔導書を読むなどの特別な経験を通してのみ上昇する。 |
| 芸術（分野） | 5      | 特定の芸術分野（絵画、音楽、彫刻など）に関する知識や実践能力。                                                |
| 経理         | 10     | 会計処理や帳簿の知識、財務状況を分析する能力。                                                                |
| 考古学       | 1      | 遺跡や遺物を通じて過去の文化や歴史を研究する知識。                                                            |
| コンピュータ | 1      | コンピューターの操作、プログラミング、ハッキングなどに関する知識や技術。                                          |
| 心理学       | 5      | 人間の行動や心理を読み取り、分析する能力。相手の意図や感情を推測する。                                |
| 人類学       | 1      | 人間の社会や文化、習俗に関する知識。異文化理解やフィールドワークに役立つ。                                  |
| 生物学       | 1      | 生物の構造、機能、生態に関する知識。                                                                        |
| 地質学       | 1      | 地球の構造、岩石、鉱物、地層に関する知識。                                                                      |
| 電子工学     | 1      | 電子回路や電子機器に関する知識や技術。                                                                          |
| 天文学       | 1      | 天体や宇宙に関する知識。                                                                                     |
| 博物学       | 10     | 自然界（動物、植物、鉱物など）に関する広範な知識。特に自然観察に基づく知識。                    |
| 物理学       | 1      | 物理法則（力学、熱、光、電気など）に関する知識。                                                          |
| 法律         | 5      | 法律や法制度に関する知識。                                                                               |
| 薬学         | 1      | 薬物の効果、調合、識別に関する知識。                                                                     |
| 歴史         | 20     | 過去の出来事、人物、時代に関する知識。                                                                      |

**注記:**
*   上記の説明は、クトゥルフ神話TRPG（主に6版）における一般的な解釈です。ルールブックの版やキーパーの裁定によって、詳細な扱いが異なる場合があります。
*   技能名の後の括弧書き（例：運転、芸術、母国語など）は、キャラクター作成時に具体的な内容を指定する必要があることを示しています。
*   回避の初期値はキャラクターのDEX（敏捷性）によって変動します。
*   母国語の初期値はキャラクターのEDU（教育）によって変動します。
